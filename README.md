## التثبيت المحلي

```bash

git clone <repository-url>

pip install -r requirements.txt

# تشغيل التطبيق

```

الموقع سيكون متاحاً على: `http://localhost:21910`

## النشر على Render

### الخطوات:

1. **إنشاء حساب على Render**: اذهب إلى [render.com](https://render.com) وأنشئ حساباً جديداً

2. **ربط مستودع GitHub**:
   - ادفع الكود إلى GitHub
   - في Render، اختر "New +" ثم "Web Service"
   - اختر مستودعك من GitHub

3. **إعدادات النشر**:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `gunicorn --bind 0.0.0.0:$PORT app:app --timeout 120`
   - **Environment**: اختر Python 3.11

4. **متغيرات البيئة** (اختياري):
   ```
   SERVER_PORT=5000
   ```

5. **انقر على Deploy** وانتظر انتهاء النشر

## استخدام بوتات تلجرام

### بدء بوت جديد:

1. سجل الدخول كمسؤول
2. اذهب إلى لوحة التحكم
3. أضف بوت جديد برقم التوكن الخاص به
4. انقر على "تشغيل البوت"

### الرابط API لتشغيل البوت:

```bash
curl -X POST http://localhost:21910/api/telegram/start \
  -H "Content-Type: application/json" \
  -d '{
    "token": "YOUR_BOT_TOKEN",
    "name": "my_bot"
  }'
```

### الحصول على قائمة البوتات:

```bash
curl http://localhost:21910/api/telegram/list
```

### إيقاف بوت:

```bash
curl -X POST http://localhost:21910/api/telegram/stop \
  -H "Content-Type: application/json" \
  -d '{"name": "my_bot"}'
```
